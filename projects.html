Everything Calculator

This application aims to develop a simulation of multiple-use calculators. There is a finance calculator that tracks your transactions and an investment calculator that calculates expected profit. The site allows a user to make an account and store data inputs. Deployed using Heroku. Heroku's free service is no longer available so the site is no longer active.

Lessons Learned

This was my first time working in a team to deliver a full stack development project. In addition, this was my first time using Django, using it with Python. I also first learned how to use GitHub, and learned the importance of communicating with a team about who is working on what issues in specific parts of the code.

Tools Used

Python, Django, Bootstrap 4, Heroku

CPU Scheduling

This application aims to develop a simulation of single-core and multi-core systems and scheduler for the cpu scheduling algorithms first-come first-served (FCFS), round-robin (RR), non-preemptive shortest job first (NSJF), and preemptive shortest job first (PSJF).

Lessons Learned

I learned about how important time and team management is in a project. I had to balance the team as well as being the Lead Developer. I had to learn how to deal with difficult partners and expressing what I and the project required of them for us to meet our goal.

Tools Used

Java, Threads, Github
